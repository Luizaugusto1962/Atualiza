# Stack: Atualiza

**Last updated:** 2026-07-06

## Language & Runtime

- **Language:** Bash (shell script), targeting Bash 4.0+
- **Runtime:** GNU Bash, with `set -euo pipefail` for strict error handling
- **Umask:** `077` (secure default file creation mask)
- **Entry point:** `atualiza.sh` (74 lines) — argument router that delegates to modules

## Configuration

- **Main config:** `configuracoes/.config` — generated by `setup.sh`, validated against command injection
- **Passwords:** `configuracoes/.senhas` — SHA-256 hashed, permission `0600`
- **Version tracking:** `configuracoes/.versao`
- **File index:** `configuracoes/indexar` — list of `.dat` files for rebuild operations
- **Temp patterns:** `configuracoes/limpetmp` — glob patterns for temp file cleanup
- **Manual:** `configuracoes/manual.txt` — full system documentation (2039 lines)

## IsCOBOL Integration

- **Supported versions:** 2018, 2020, 2023, 2024, 2025, 2026
- **File types managed:** `.class` (compiled), `.mclass` (debug), `.TEL` (screens), `.so` (shared objects)
- **Library format:** `tar.gz` archives of `E_EXEC` and `T_TELAS` directories
- **Rebuild tool:** `jutil -rebuild <file> -a -f`
